# Sprint 003 — L10n End-to-End

Tujuan: menghubungkan locale switching (yang sudah berjalan di Sprint 002) ke UI yang sesungguhnya — semua teks hardcode di app diganti dengan key `AppLocalizations`, sehingga mengganti bahasa di Settings langsung mengubah tampilan teks seluruh app.

Acceptance criteria sprint: ganti bahasa ke English di Settings → semua teks UI berubah ke EN tanpa restart. Ganti kembali ke Indonesia → teks kembali ke ID. Preferensi tersimpan antar sesi.

> **Catatan:** Locale switching via `localeNotifierProvider` dan wiring ke `MaterialApp.router` sudah selesai di Sprint 002. Sprint ini hanya menambah key L10n dan mengganti teks hardcode.

---

## Arsitektur L10n

File ARB sebagai source of truth teks berada di `packages/core`:

```text
packages/core/lib/src/l10n/
├── app_id.arb     ← Bahasa Indonesia (default)
└── app_en.arb     ← English
```

Generated class `AppLocalizations` diakses dari UI via:

```dart
final l10n = AppLocalizations.of(context)!;
Text(l10n.someKey);
```

Key baru ditambahkan ke **kedua** file ARB secara bersamaan, lalu generate ulang:

```bash
cd packages/core
flutter gen-l10n
```

## Phase 2 — Wire L10n ke `apps/main`

- [x] Update `apps/main/lib/home/home_screen.dart` — ganti teks hardcode:
  ```dart
  import 'package:flutter/material.dart';
  import 'package:go_router/go_router.dart';
  import 'package:core/core.dart'; // AppLocalizations sudah diexport dari core

  class HomeScreen extends StatelessWidget {
    const HomeScreen({super.key});

    @override
    Widget build(BuildContext context) {
      final l10n = AppLocalizations.of(context)!;
      return Scaffold(
        appBar: AppBar(
          title: Text(l10n.home),
          actions: [
            IconButton(
              icon: const Icon(Icons.settings),
              onPressed: () => context.push(AppRoutes.settings),
            ),
          ],
        ),
        body: Center(child: Text(l10n.welcome)),
      );
    }
  }
  ```
- [x] Update `apps/main/lib/features/settings/presentation/settings_screen.dart` — ganti semua teks hardcode:
  - `'Settings'` → `l10n.settings`
  - `'Appearance'` → `l10n.appearance`
  - `'System default'` → `l10n.themeSystem`
  - `'Light'` → `l10n.themeLight`
  - `'Dark'` → `l10n.themeDark`
  - `'Language'` → `l10n.language`
  - `'Indonesia'` → `l10n.langIndonesia`
  - `'English'` → `l10n.langEnglish`
  - `'Version'` → `l10n.version`
  > `_ThemeTile`, `_LanguageTile`, `_SectionHeader`, `_AboutTile` adalah `StatelessWidget` — akses `l10n` via `AppLocalizations.of(context)!` di dalam `build()`.
- [x] `dart analyze apps/main` — no issues
